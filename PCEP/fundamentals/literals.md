## Literals

Literal Types:
    1. Integers
        * Octal numbers
        * Hexadecimal numbers
    2. Floating point numbers
    3. Strings
    4. Booleans
```
>>> print(type(45.50))
<class 'float'>
>>> print(type(1_00_0000_000))
<class 'int'>
```

Booleans:
    1. true (1)
    2. false (0)

String literals 
- can be created by writing a text(a group of characters) surrounded by the single(") or double(" ")quotes

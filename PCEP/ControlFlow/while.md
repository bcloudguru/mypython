## While - Conditional Statements
> code block is repeated until conditions are met

```
>>> secret_number = 3
>>> guess = int(input("guess a number: " ))
guess a number: 5
>>> while guess != secret_number:
...     guess = int(input("guess a number: " ))
... 
guess a number: 6
guess a number: 7
guess a number: 3
```
